# Meme Stream

A meme stream application is to have the fun with the memes.

## Tech Stack(MERN)

- MongoDB
- ExpressJS
- ReactJS
- NodeJS

## App

You can install the software requirements using the `install.sh` script file. And run the server using `server_run.sh` file.
